# Site Institucional — ACIERJ

Site institucional da **Associação dos Cuidadores da Pessoa Idosa, da Saúde Mental e com Deficiência do Estado do Rio de Janeiro (ACIERJ)**.

O projeto tem como objetivo criar uma plataforma digital clara, acessível e institucionalmente sólida para apresentação da associação, divulgação de suas ações, produção de conteúdo informativo e facilitação de contato com cuidadores, parceiros e a sociedade.

---

## 🎯 Objetivos do Projeto

- Construir um site institucional moderno, acessível e responsivo.
- Comunicar de forma clara a missão, os valores e as áreas de atuação da ACIERJ.
- Servir como canal de informação, apoio e orientação para cuidadores e familiares.
- Desenvolver um projeto real para portfólio técnico em desenvolvimento web.

---

## 🧠 Escopo Inicial

- Página inicial (Home)
- Quem Somos
- Áreas de Atuação
  - Pessoa Idosa
  - Saúde Mental
  - Pessoa com Deficiência
- Projetos e Ações
- Blog / Conteúdos
- Contato

---

## 🛠️ Tecnologias Utilizadas

- Next.js (Pages Router)
- React
- HTML5 semântico
- CSS3 (Flexbox / Grid)
- Git e GitHub para versionamento

---

## 🚀 Como executar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Abra [http://localhost:3000](http://localhost:3000) no navegador.
